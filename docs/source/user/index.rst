==========
User Guide
==========

This guide provides an overview of arim's functionality, as well as some background to ultrasonic imaging where advanced
concepts are used. For more information about ultrasonic inspection, imaging, and nondestructive testing in general,
users are referred to the :ref:`bibliography`.

If you would like to jump straight in, a set of examples have been provided to demonstrate arim's functionality. For an
overview, see :ref:`examples`.

Below, a range of guides to the functionality are provided. More details may be found in the :ref:`api`.
